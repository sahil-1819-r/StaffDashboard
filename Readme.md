# 👨‍💼 Staff Queue Management Dashboard

A web-based staff dashboard designed to manage customer queues efficiently. The dashboard allows staff members to generate tokens, monitor live queue status, manage customer flow, and handle unavailable customers using a hold-and-recall mechanism.

---

## 📌 Overview

This project provides a simple and efficient interface for staff members to manage customer queues digitally. It eliminates the need for manual queue tracking and helps maintain a smooth service flow through token generation, queue monitoring, and customer status management.

---

## ✨ Features

### 🎟 Generate Tokens
- Add customers manually.
- Automatically generate unique token numbers.
- Maintain a structured queue system.

### 📋 Live Queue Monitoring
- View all generated tokens.
- Track customer names and statuses.
- Real-time queue updates.

### 📞 Call Next Customer
- Serve customers based on queue order.
- Automatically update the currently active token.

### ⏸ Hold Customer
- Place customers on hold if they are unavailable.
- Continue serving the next customers without disrupting the queue.

### 🔄 Recall Held Customers
- Bring customers back from the hold list.
- Resume service once they return.

### 📊 Analytics Dashboard
Track queue performance with:
- Total Tokens Generated
- Customers Served
- Pending Customers

## 🖥️ Dashboard Sections

### Add Person
Allows staff to enter a customer's name and generate a queue token.

### Current Token
Displays the customer currently being served.

### Hold List
Stores customers who are temporarily unavailable.

### Analytics
Provides real-time queue statistics and performance metrics.

### Live Queue
Displays all generated tokens along with customer details and status.
